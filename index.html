Github Pagesとは

Githu Pagesとは、GithubのブランチでWebページを作れる機能!

作り方

  • リポジトリ上にgh-pagesってリポジトリを作成する
  • ブランチを移動する。
  • ブランチ上に作成されたファイルは http://(ユーザー名).github.io/(リポジトリ名) でアクセスできるようになる。
    このページはOrganizationでできているので、URLはhttp://SoftwareGroup2.github.io/git_tutorial/みたいになると思う。。。
    初回はしばらく時間がかかるので注意。
  • index.htmlがディレクトリにアクセスした時に表示される(サーバーの設定依存)

注意

  • phpなどのCGIを動かすことはできない
  • 更新まで時間がかかる
  • ブランチはmasterではなくgh-pages

github pagesにブランチで作成したプログラムの説明とかあると、むっちゃカッコイイ!!!
